# Coursework Data

The local submission package includes the raw coursework data files in this
folder so that `notebooks/c2o_research_process_and_tests.ipynb` can be run
top-to-bottom without moving files.

If this repository is cloned from GitHub, the large raw data files may be absent
because they are intentionally not tracked there. In that case, copy the
coursework data package into this folder before running the notebook.

Expected files:

- `all_data.parquet`
- `cheapness_scores.parquet`
- `earnings_calendar.parquet`
- `earnings_transfo.parquet`
- `gics_info.parquet`
- `piotrosky.parquet`
- `prices.parquet`
- `regime.parquet`
- `rolling_scores_downgrade.csv`
- `rolling_scores_upgrade.csv`
- `short_interest_transfo.parquet`
- `sp400_constituents.parquet`
- `sp500_constituents.parquet`
- `sp500_tr.parquet`
